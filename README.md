# README

## About

This is a devise :user template that incorporates the latest bootstrap 4 beta

* Rails 5.1.4
* Devise 4.3
* bootstrap 4.beta.2

## Setup

* Clone the repository
```
    git clone https://github.com/simonireilly/redesigned-tempalte4.git
```
* Setup the database
```
    rails db:migrate
```
* Serve the application
```
    rails s
```
